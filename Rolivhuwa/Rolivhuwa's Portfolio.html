Project 2

Coding is Love

In my portfolio, I have a project called "Movie Recommend", designed to help users discover movies based on their preferred genre. Here's a breakdown of how it works: Genre Selection: The user is first presented with a dropdown list of genres (e.g., Action, Comedy, Drama, etc.). Movie Recommendation Dropdown: After selecting a genre, another dropdown appears, allowing the user to refine their search by selecting a specific type of movie within that genre. For example, in the Action genre, they could choose between "Superhero", "Adventure", or "Thriller". Dynamic Recommendations: Based on the user's selection, the system recommends a list of movies that fit both the chosen genre and subcategory.

Project 3

Coding is Love

The Tic Tac Toe project is a simple but engaging game developed using HTML, CSS, and a bit of JavaScript. The game allows two players to compete—one playing as "X" and the other as "O." Here's how it works: Gameplay: Players take turns placing their respective symbols ("X" or "O") on a 3x3 grid. The objective is to get three of your symbols in a row, column, or diagonal. Win Conditions: When a player successfully aligns three of their symbols (either "X" or "O"), a message pops up declaring the winner, for example, "X won." Draw Condition: If all the cells on the grid are filled without any player winning, the game ends in a draw, and a message like "It's a draw!" is displayed.
